## 🚀 Quick Start

### Installation Steps:

1. **Go to Releases** - Navigate to the [Releases](../../releases) section of this repository
2. **Download the Game** - Download `Game.rar` from the latest release
3. **Extract Files** - Unpack the `main` folder to your desired location
4. **Navigate to Folder** - Open the extracted `main` folder
5. **Run the Game** - Find and double-click `main.exe`
6. **Play!** - Enjoy the game! 🎮
